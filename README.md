Syntecxhub Python Student Management System 🎓
A lightweight Command Line Interface (CLI) application built in Python to manage student records. This project demonstrates foundational Object-Oriented Programming (OOP) concepts, basic data validation, and file I/O operations.

✨ Features
Add Students: Create new student records with an ID, Name, and Grade.
Update Records: Modify a student's name or grade easily.
Delete Students: Remove student records using their unique ID.
List All Students: View all records in a cleanly formatted console table.
Data Persistence: Records are automatically saved to and loaded from a local students.json file.
Data Validation: Prevents the creation of duplicate student IDs.
🛠️ Technologies Used
Python 3: Core application logic.
JSON: Used for persisting student data locally.
OOP Principles: Utilizes Student and StudentManager classes to encapsulate data and operations.
🚀 How to Run
Make sure you have Python installed on your computer.
Download or clone this repository.
Open your terminal or command prompt in the project folder.
Run the following command: python student_management.py
